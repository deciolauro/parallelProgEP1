Detailed Description

Simple parser script to extract all the relevant features.

This Python parser extracts all the relevant features from all .log files produced by the execution of run_measurements.sh script and dumps the information in a prickle file named finaldata.p to be used by the plotter script

Usage: python parser.py

Notes:

  • This file and all the .log files should be placed in the same directory
